# Laravel2IPFS
Laravel2IPFS is a plug-in that enables applications developed with Laravel infrastructure to be backed up to IPFS. There is also a feature that will allow you to quickly restore from your backups in case of any disaster.

* <b>How are you planning to improve this project?</b><br>
I will develop a plugin for the project that will initially back up the database, design files and media files for the Laravel framework. For this plugin, I will initially use the APIs of IPFS infrastructure providers such as Infura. After developing the backup module, I will develop the one-click install module from the backup to a fresh Laravel installation. After that, I'm thinking of developing a base module on PHP. Thanks to this module, me and other developers will be able to integrate the module into any framework they want.
